# Spatial Canvas v3

Three.js を使った 3D 空間ブレインストーミングツール。付箋（ノート）を 3D 空間上に配置・整理し、GitHub Issues としてエクスポートできます。

## 特徴

- **3D 空間にノートを配置** — ドラッグで自由に移動、カメラを回転・ズームして俯瞰
- **5 層レイヤー管理** — Vision / Strategy / Feature / Design / Implement の Z 軸階層
- **複数ビューモード** — 3D 自由視点、Category × Priority、Depth × Priority、Depth × Category
- **フリーハンド描画** — 3D 空間上に線を描いてノート間の関係を可視化
- **フォーカスモード** — Z 軸スライダーで特定レイヤーにフォーカス
- **GitHub Issues エクスポート** — ノートを Issue 形式に変換（ラベル・Assignee・Milestone 付き）
- **デモウォークスルー** — ステップバイステップのガイド付きツアー

## 使い方

`spatial-canvas-v3.html` をブラウザで開くだけで動作します。外部依存はありません。

### キーボードショートカット

| キー | 操作 |
|------|------|
| `N` | ノート追加 |
| `D` | 描画モード切替 |
| `F` | フォーカスモード切替 |
| `1` / `2` / `3` / `4` | ビュー切替（3D / XY / ZY / ZX） |
| `Ctrl+Z` | 元に戻す |
| `Esc` | モード解除・全体ビューに戻る |

## 技術構成

- **Three.js**（インライン埋め込み）
- **OrbitControls** によるカメラ操作
- 単一 HTML ファイルで完結
